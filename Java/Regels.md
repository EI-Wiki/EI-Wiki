## Algemene regels in Java:
1. Naam van een klasse:
  De naam van een klasse in Java moet altijd met een hoofdletter beginnen.
  
    Voorbeeld:
    
      ``public class MyClass{...}``
  
    Dus niet:
  
      ``public class myclass{...}``
